Framework

Framework er standardiserede kodeelementer, der avendes til opbygning af layouts.
Det giver mulighed for ikke at skulle lave alt arbejde selv og starte forfra hver gang.

Der er både fordele og ulemper ved at bruge framework templets, før man går i gang er det
derfor en god ide at sætte sig ind i det, for det er noget man skal bruge tid på at lærer.

Bootstrap

Bootstrap er en af mange framework der findes.
Bootstrap kan gøre designet responsiv, der er 4 forskellige breakpoints:

  • xs: mobiler
  • sm: tablet
  • md: desktop
  • lg: stor desktop
