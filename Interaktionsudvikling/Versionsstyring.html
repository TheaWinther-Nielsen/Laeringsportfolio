Versionsstyring


Versionsstyring giver en mulighed for at gå tilbage til tidligere versioner og fortage nogle ændringer. Man kan også være flere om at komme med forslag eller kommentar til ændringer i koden samt evt. fejl det kan være en stor fordel, da man kan se sig blindt på sit eget arbejde og man kan samtidigt være flere om at arbejde på samme projekt/filer. Det kan være relevant at bruge for så kan man få input fra andre som evt. er mere erfarende indenfor området. Her kan man få både hjælp og inspiration. Bruges til at holde styr på ændringer fortaget i ens projekt filer.

Git & Github

Git er et versionsstyringssystem, der skrevet af Linus Thorvald til at holde styr på udviklingen af Linux. På webplatformen GitHub kan man oprette Git-repositories. GitHub er en platform hvor man kan uploade sin kode og dele den med andre, man kan vise hvad man kan og hvad man har lavet. Når man har uploadet sin kode på GitHub kan andre komme med ændringer og kommentar. GitHub er en slags socialmedie for mennesker der sidder og arbejder med kode. GitHub er både bruges direkte online eller det kan downloades til computeren GitHub desktop.
